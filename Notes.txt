State Management :

- Preserve the state of the web page / application / object / data

Client side sm :happens in the browser / client computer
Server side sm : stores the data in the server side 
Cookies : piece of information that is stored in the browser
- easily tampered \ delete a cookie
- do not use sensitive info on the cookies 


Why ?

Http - Hyper text transfer protocol
- stateless protocol

State management techniques - Client side :

- cookie

Store data in the browser

Query String :
 
 - when you pass data from one page to another page over the URL

 - denoted by the ? Symbol followed by the data

 Hidden field :

 - Hide your data on client

 - maintain the data when sending from controller to view
 - Dictionary object (key,value pair)
 - ViewDataDictionary (base class)
